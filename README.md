# Parth Kevadiya — Portfolio

> **Live:** [parthkevadiya.com](https://www.parthkevadiya.com)

Personal portfolio for **Parth Kevadiya** — AI Engineer & Full-Stack Developer based in Canada, specializing in backend systems, cloud AI, machine learning, RAG applications, and LLMOps.

---

## Tech Stack

| Layer | Technology |
|-------|-----------|
| Framework | React 19 + TypeScript + Vite |
| Styling | Tailwind CSS 3.4 + custom CSS |
| Animation | Three.js, Motion, GSAP, Lenis |
| UI | Radix UI, Embla Carousel, Framer Motion |
| Fonts | Playfair Display, IBM Plex Mono, Inter |

---

## Features

- **Cinematic hero** — scroll-driven video frame sequence with text overlays
- **Three.js wave background** — animated particle grid with theme-aware colors
- **Expand-on-hover project cards** — proportional flex layout
- **Work experience** — interactive stack interactor
- **Skills grid** — logo carousel
- **Social card** — nested stacked layer design (GitHub, LinkedIn, Instagram, Discord)
- **Contact form** — animated form with validation
- **SEO** — JSON-LD Person schema, Open Graph, Twitter cards, sitemap, robots.txt

## Project Structure

```
portfolio/
├── components/ui/        # React island components
│   ├── dotted-surface.tsx      # Three.js wave bg
│   ├── expand-cards.tsx        # Project cards
│   ├── logos3.tsx              # Skills carousel
│   ├── connoisseur-stack-interactor.tsx  # Work experience
│   ├── contact-form.tsx        # Contact + socials
│   ├── social-card.tsx         # Nested social card
│   └── loader.tsx              # Preloader cube
├── public/
│   ├── links.html              # /links page (SEO)
│   ├── sitemap.xml
│   └── robots.txt
├── src/main.tsx          # React island mounts
├── index.html            # HTML shell + meta/SEO
├── styles.css            # Global styles
└── script.js             # Scroll/motion logic
```

## Getting Started

```bash
npm install

npm run build     # Production build → dist/
npm run preview   # Preview production build
```
